# kyouen
共円とは4点が同一円周上に存在しないように点を打っていくゲームです．<br>
詳細は[こちら](https://www.google.com/search?q=%E5%85%B1%E5%86%86+%E3%82%B2%E3%83%BC%E3%83%A0)からどうぞ

# ルール
2人向け<br>
持ち時間1分+20秒/1手 フィッシャールール<br>
共円の宣言にはページ下部の"kyouen!"ボタンを押してから，残りの3点を選択してください
